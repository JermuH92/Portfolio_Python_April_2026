# Python-Learning Path: 5 project bundle 

This repository contains my first steps with Python programming as part of the Boot.dev learning path. 
The projects have been composed as a singular bundle that represents my development from introductory Python to Object-Oriented (OOP) and Functional Programming (FP), as well as briefly touching on the use of APIs.

** NOTE: ** These projects were built independently of the Boot.dev course material to reinforce the skills learned during the course and solidify core concepts by building original projects from scratch.

---

## 🛠️ General Requirements
- **Python:** 3.x
- **Libraries:** `requests` (for API use in crypto_bot)
- **Virtual Environments:** Recommended to use `.venv`-folder for dependency control.

---

## 📂 Projects and learned concepts

### 1. Password Checker (predicate based)
First project, focusing on programming logic and string manipulation.
- **Concepts solidified:** `if/else`-conditional structures, `for`-loops and using predicate based functions to return True or False if password follows predicate guidelines.

### 2. Recursive File Counter
A tool that traverses directory structures and analyzes file counts.
- **Concepts solidified:** `os` library usage, file system navigation, and conceptualizing hierarchical directory structures.

### 3. Data Pipeline
Data handling tool utilizing Functional Programming principles.
- **Concepts solidified:** JSON-data read/write operations, and the use of `map`, `filter` ja `reduce` -functions (FP).

### 4. Flashcards CLI
Interactive command-line interface game for flashcard-style studying.
- **Concepts solidified:** Object-Oriented Programming (OOP), Classes (`Card`, `Deck`), class methods, state management, and building a robust game loop.

### 5. Crypto Tracker Bot (Binance API)
An API-driven tool using real-time market data for Bitcoin.
- **Concepts solidified:** Fetching external data with the `requests` library, HTTP status codes, JSON response parsing, and error handling (`try/except`).

---

## 🚀 How to run the programs?
1. Clone the repository: `git clone [personal-repo-url]`
2. Navigate to the project folder: `cd [project-name]`
3. Create and activate a virtual environment
4. (For Crypto Bot) Install dependencies: `pip install requests`
5. Run the program: `python3 filename.py`
